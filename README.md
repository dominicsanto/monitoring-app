## Installation
* Clone or download the repo

  `git clone git@github.com:dominicsanto/monitoring-app.git`

* Install node modules within directory

  `npm install`

* Run the application

  `npm run start`

## Tech Stack
1. [ElectronJS](https://www.electronjs.org/) - The Electron Framework
